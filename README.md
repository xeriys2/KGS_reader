# KGS_Reader

Настольное приложение (Tkinter) для пакетной обработки PDF‑файлов геодезических съёмок КГС.

## Возможности

- Выбор папки с PDF и удобный список файлов (сортировка, поиск, массовый выбор, ПКМ).
- Извлечение текста из PDF через PyMuPDF (fitz); при отсутствии текстового слоя — OCR через Tesseract.
- Автоматическое извлечение полей: `Тип коммуникации`, `Номер договора`, `КГС`, `Дата съёмки`.
- Опциональный импорт таблиц координат точек и сохранение их в TXT.
- Формирование Excel‑реестра `Реестр_геодезических_съемок.xlsx`.
- Прогресс‑бар, счётчики файлов/страниц, кооперативная отмена.
- Редактор типов коммуникаций (сохраняется в `comm_types.json`).

## Запуск из исходников

Требуется Python 3.9+.

Установка зависимостей:

```bash
pip install pymupdf pytesseract pillow openpyxl
```

Запуск:

```bash
python "KGS_Reader v6.py"
```

Для OCR нужен установленный/портебл Tesseract.

## Портебл‑сборка

Для портебл версии обычно используется PyInstaller `--onedir` и вложенный Tesseract:

```
KGS_Reader_Portable/
  KGS_Reader v6.exe
  tesseract/
    tesseract.exe
    tessdata/
      rus.traineddata
      eng.traineddata
```
